NOTE: HyperNews supports another subscription level called "notify" which is whether or not you are emailed when someone posts a reply to one of your messages. It is possible for other members to add themselves to this "notify" list for any particular message. This is called "subthread subscription". Subthread subscription status cannot be changed here. You must go to the Subscribe page for the desired message. Use the "Forums and Subthreads" option below to find ALL the messages in the HyperNews system for which you will be "notified".
